# Uygulama ile İlgili Bilgiler

## Uygulama İsmi
Gidelim

## Uygulama Açıklaması
Gidelim, kullanıcıların bulundukları konuma göre restoran, kafe ve diğer yeme-içme mekanlarını keşfetmesini sağlayan bir mobil uygulamadır. Kullanıcılar mekanları arayabilir, detaylarını görüntüleyebilir ve gitmek istedikleri yerleri planlananlar listesine ekleyebilir.

Uygulama modern iOS mimarileri kullanılarak geliştirilmiştir. MVVM yapısı, dependency injection yaklaşımı ve async/await tabanlı veri akışı sayesinde ölçeklenebilir ve test edilebilir bir yapı sunar. Backend tarafında kimlik doğrulama ve veri yönetimi için Firebase servisleri kullanılmaktadır.

Proje, temiz kod prensipleri, kapsamlı unit test yazımı ve test edilebilir mimari yaklaşımı ile üretim ortamına uygun olacak şekilde tasarlanmıştır. Kodlar versiyon kontrolü ve ekip iş birliği için GitHub üzerinde paylaşılmaktadır.

## Hedef Kitle
* Yeni mekanlar keşfetmeyi seven kullanıcılar
* Bulunduğu bölgede restoran / kafe önerisi arayan kişiler
* Gitmek istedikleri mekanları önceden planlamak isteyen kullanıcılar
* Seyahat eden ve gittiği şehirde hızlıca mekan bulmak isteyen kullanıcılar
